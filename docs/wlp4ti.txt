WLP4 Typed Intermediate Format (.wlp4ti)
========================================

A `.wlp4ti` file is a `.wlp4i` parse tree with **type annotations** on expression-related nodes, produced after semantic analysis. Traversal is the same preorder as `.wlp4i`.

Annotation rules
----------------

**Terminals:** After `KIND lexeme`, if the kind is `NUM` or `NULL`, or if the kind is `ID` and the identifier is a **variable** (not a procedure name), append ` : long` or ` : long*`. Then newline.

**Nonterminals:** Print the production line; if the LHS is `expr`, `term`, `factor`, or `lvalue`, append ` : long` or ` : long*`. Then newline; recurse on children left to right.

Example
-------

Program:

  long wain(long* start, long size) {
    long* end = NULL;
    end = start + size;
    while(start < end) { println(*start); }
    return 241;
  }
